## 📑 슬라이드 3: 힙(Heap)이란?
# 힙(Heap)이란?

### 🎯 기본 정의
- **완전 이진 트리(Complete Binary Tree)** 형태의 자료구조
- **힙 속성(Heap Property)**을 만족해야 함
- **우선순위 큐**를 효율적으로 구현하기 위해 설계됨

### 🌳 완전 이진 트리란?
- 마지막 레벨을 제외한 모든 레벨이 완전히 채워진 트리
- 마지막 레벨은 **왼쪽부터 순서대로** 채워짐
- 배열로 표현하기에 최적화된 구조

### 💡 힙의 핵심 아이디어
- **부모-자식 간 순서 관계**를 유지
- **완전 이진 트리**로 균형 잡힌 구조 보장

## 📑 슬라이드 4: Min-Heap vs Max-Heap
# 힙의 두 가지 종류

### 🔻 Min-Heap (최소 힙)
- **힙 속성**: `부모 노드 ≤ 자식 노드`
- **루트**: 전체에서 **가장 작은 값**
- **용도**: 최솟값을 빠르게 찾기

```
       1        ← 최솟값 (루트)
     /   \
    3     2
   / \   / \
  4   7 5   6
```

### 🔺 Max-Heap (최대 힙)
- **힙 속성**: `부모 노드 ≥ 자식 노드`  
- **루트**: 전체에서 **가장 큰 값**
- **용도**: 최댓값을 빠르게 찾기

```
       10       ← 최댓값 (루트)
     /    \
    7      9
   / \    / \
  3   4  6   8
```

## 📑 슬라이드 5: 시간복잡도 & 특징
# 성능 특징

| 연산 | 시간복잡도 | 설명 |
|------|------------|------|
| **최솟값 조회** | `O(1)` | 루트 노드 접근 |
| **삽입** | `O(log n)` | Bubble Up |
| **삭제** | `O(log n)` | Bubble Down |
| **힙 생성** | `O(n)` | Heapify |

### 💡 언제 사용하나?
- 우선순위 큐, 힙 정렬
- 다익스트라 알고리즘
- Top-K 문제 해결

- **오른쪽 자식**: `2*i + 2`

## 📑 슬라이드 8: 삽입 과정 (2/2)
# Bubble Up 과정

### 2단계: 부모와 비교 & 교환
```
0 < 4 → 교환
0 < 3 → 교환  
0 < 1 → 교환
```

### 최종 결과
```
       0
     /   \
    1     2
   / \   / \
  3   7 5   6
 /
4
```
**시간복잡도**: `O(log n)` - 트리 높이만큼 이동

---

## 📑 슬라이드 9: 삭제 과정 (1/2)
# 삭제 과정: 최솟값 제거

### 1단계: 루트 제거 & 마지막 노드를 루트로
```
삭제 전:           삭제 후:
   0                 4
 /   \             /   \
1     2    →      1     2
```

---

## 📑 슬라이드 10: 삭제 과정 (2/2)
# Bubble Down 과정

### 2단계: 자식들과 비교 & 교환
```
4 > min(1,2) → 1과 교환
4 > min(3,7) → 3과 교환
```

### 최종 결과
```
       1
     /   \
    3     2
   / \   / \
  4   7 5   6
```
**시간복잡도**: `O(log n)`

## 📑 슬라이드 11: Python 구현 - 기본 사용법
# Python heapq 모듈

```python
import heapq

# 힙 생성 및 삽입
heap = []
numbers = [4, 1, 3, 2, 16, 9, 10]

for num in numbers:
    heapq.heappush(heap, num)

print(heap)  # [1, 2, 3, 4, 16, 9, 10]

# 최솟값 제거
min_val = heapq.heappop(heap)
print(min_val)  # 1
```

## 📑 슬라이드 12: Python 구현 - 고급 활용
# 실무 활용 예제

### 🔍 Top-K 원소 찾기
```python
# 가장 작은 3개
heapq.nsmallest(3, [7,1,4,2,9])  # [1, 2, 4]

# 가장 큰 3개  
heapq.nlargest(3, [7,1,4,2,9])   # [9, 7, 4]
```

### 🏗️ 배열을 힙으로 변환
```python
arr = [4, 1, 3, 2, 16, 9, 10]
heapq.heapify(arr)  # O(n) 시간!
```
